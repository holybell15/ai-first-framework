# US_F01_CreateTask

## User Story

As a user, I want to create a task with a title, so that I can record what I need to do.

## Acceptance Criteria

1. When the user enters a title and submits, a new task is created.
2. When the title is empty, the form blocks submission and shows an error.
3. After creation succeeds, the new task appears in the task list.
