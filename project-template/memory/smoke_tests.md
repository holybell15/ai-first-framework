# [專案名稱] Prototype 冒煙測試清單（Smoke Tests）

> **用途**：每次 CR（改功能）後，必須逐一手動驗證以下項目，確認原有功能未被破壞。
> **頻率**：任何 Edit 完成後、回報給負責人前，Claude 自己先跑一遍確認。
> **格式**：✅ 通過 / ❌ 失敗（需修復）/ ⚠️ 部分正常

---

## 使用說明

每次 Claude 完成 Edit 後，在 response 中附上以下格式的冒煙測試結果：

```
### 🧪 冒煙測試快速驗收
- [測試編號] [測試項目]：✅
- [其他受本次改動影響的項目]：✅
- 其餘項目：未改動範圍，前版已驗證
```

若 Claude 無法親自執行瀏覽器測試，應明確標注「⚠️ 需人工確認」。
