# [專案名稱] API 錯誤碼標準 v1.0

> 版本：1.0 | 建立：2026-03-15

## 錯誤碼格式

```
{PREFIX}-{LAYER}{CODE}
```

- **PREFIX**：`[專案代號]`（使用前替換為實際專案縮寫，例如 AICC、CRM、HRIS）
- **LAYER**：功能層級
- **CODE**：3 位數字

## 層級定義

| Layer 代碼 | 說明 |
|-----------|------|
| `A` | 認證/授權 (Authentication/Authorization) |
| `B` | 業務邏輯 (Business Logic) |
| `D` | 資料庫 (Database) |
| `I` | 整合/外部服務 (Integration) |
| `S` | 系統/基礎設施 (System) |
| `V` | 驗證 (Validation) |

## 標準錯誤碼

### A — 認證/授權

| 錯誤碼 | HTTP Status | 說明 |
|--------|-------------|------|
| [PREFIX]-A001 | 401 | Token 無效或過期 |
| [PREFIX]-A002 | 401 | 未提供認證資訊 |
| [PREFIX]-A003 | 403 | 權限不足 |
| [PREFIX]-A004 | 403 | 跨租戶存取被拒 |

### V — 驗證

| 錯誤碼 | HTTP Status | 說明 |
|--------|-------------|------|
| [PREFIX]-V001 | 400 | 必填欄位缺漏 |
| [PREFIX]-V002 | 400 | 欄位格式錯誤 |
| [PREFIX]-V003 | 400 | 欄位值超出範圍 |

### B — 業務邏輯

| 錯誤碼 | HTTP Status | 說明 |
|--------|-------------|------|
| [PREFIX]-B001 | 404 | 資源不存在 |
| [PREFIX]-B002 | 409 | 資源已存在（重複） |
| [PREFIX]-B003 | 422 | 業務規則不符 |

### D — 資料庫

| 錯誤碼 | HTTP Status | 說明 |
|--------|-------------|------|
| [PREFIX]-D001 | 500 | 資料庫連線失敗 |
| [PREFIX]-D002 | 500 | 查詢逾時 |

### I — 整合

| 錯誤碼 | HTTP Status | 說明 |
|--------|-------------|------|
| [PREFIX]-I001 | 502 | 外部服務無回應 |
| [PREFIX]-I002 | 504 | 外部服務逾時 |

### S — 系統

| 錯誤碼 | HTTP Status | 說明 |
|--------|-------------|------|
| [PREFIX]-S001 | 500 | 未預期的系統錯誤 |
| [PREFIX]-S002 | 503 | 服務暫時不可用 |

## 錯誤回應格式（JSON）

```json
{
  "error": {
    "code": "[PREFIX]-A003",
    "message": "您沒有執行此操作的權限",
    "detail": "需要 ADMIN 角色才能存取此資源",
    "timestamp": "2026-03-15T10:00:00Z",
    "traceId": "abc-123-xyz"
  }
}
```
