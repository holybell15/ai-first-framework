# Theme Factory Skill

Apply consistent, professional styling to any artifact using curated color palettes and font pairings. Whether it's a presentation deck, document, or HTML landing page — this skill makes it look polished.

## Purpose

Each theme includes:
- A cohesive color palette with hex codes
- Complementary font pairings for headers and body text
- A distinct visual identity suitable for different contexts and audiences

## Usage Instructions

To apply styling to any artifact:

1. **Show the theme showcase**: Display the `theme-showcase.pdf` file so users can see all available themes visually. Do not modify it; simply show it for viewing.
2. **Ask for their choice**: Which theme appeals to them?
3. **Wait for selection**: Get explicit confirmation about the chosen theme
4. **Apply the theme**: Once chosen, apply the selected theme's colors and fonts consistently throughout the artifact

## 10 Themes Available

Each showcased in `theme-showcase.pdf`:

1. **Ocean Depths** - Professional and calming maritime theme
2. **Sunset Boulevard** - Warm and vibrant sunset colors
3. **Forest Canopy** - Natural and grounded earth tones
4. **Modern Minimalist** - Clean and contemporary grayscale
5. **Golden Hour** - Rich and warm autumnal palette
6. **Arctic Frost** - Cool and crisp winter-inspired theme
7. **Desert Rose** - Soft and sophisticated dusty tones
8. **Tech Innovation** - Bold and modern tech aesthetic
9. **Botanical Garden** - Fresh and organic garden colors
10. **Midnight Galaxy** - Dramatic and cosmic deep tones

## Application Process

After a preferred theme is selected:

1. Read the corresponding theme file from the `themes/` directory
2. Apply the specified colors and fonts consistently throughout the artifact
3. Ensure proper contrast and readability
4. Maintain the theme's visual identity across all slides/pages

## Create a Custom Theme

For cases where none of the existing themes fit, create a custom theme. Based on the user's description, generate a new theme similar to the ones above. Give the theme a descriptive name. Choose appropriate colors/fonts based on their input. After generating, show it for review. Following approval, apply it as described above.
