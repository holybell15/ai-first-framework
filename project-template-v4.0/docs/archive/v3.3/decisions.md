# Architecture Decision Records

> Append-only. 決策可被新 ADR supersede，但不刪除舊的。

## ADR 格式

```yaml
adr_id: "ADR-{NNN}"
title: "[決策主題]"
status: "accepted | superseded | deprecated"
date: "YYYY-MM-DD"
owner: "Architect"
context: "為何需要這個決策"
decision: "最終選擇"
alternatives: "考慮過的其他選項"
consequences: "這個決策的影響"
```
